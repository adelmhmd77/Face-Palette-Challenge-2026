// القواعد والشروط

القواعد ملزمة

  • 01 لا بورتفوليو شخصي. الموقع لا يمكن أن يكون عرضاً لأعمالك أو سيرتك الذاتية. النيتش يجب أن يكون مختلفاً كلياً.
  • 02 ChatGPT يحدد فكرة الموقع. لا يمكنك تغيير النيتش الذي يقترحه — التزم بما يولّده الذكاء الاصطناعي تماماً.
  • 03 ChatGPT يحدد حجم المحتوى. عدد الكلمات لكل قسم محدد مسبقاً من البرومبت، ويجب الالتزام به دون زيادة أو نقصان.
  • 04 صفحة واحدة فقط. جميع المخرجات يجب أن تكون Landing Page واحدة — لا routing، لا صفحات إضافية.
  • 05 البرومبت يولّد كل شيء. لوحة الألوان، الأقسام، الطباعة، التخطيط، الأنيميشن، والخلفيات — كلها تأتي من الـ prompt مباشرةً.
  • 06 شبكة الألوان مطلوبة. يجب توليد grid يعرض جميع ألوان اللوحة مع كتابة كود Hex واضح على كل مربع.

// البرومبت الرسمي

انسخ وابدأ

هذا هو البرومبت الرسمي للتحدي. أرسله مع صورة وجهك إلى ChatGPT للبدء.

face-palette-challenge.prompt 
I will send you an image of my face. Analyze the colors in my face and generate a color palette for a website based on it.

**Rules:**
- The website cannot be a personal portfolio.
- You determine the website niche/concept.
- You determine content size and word count per section; it must be followed.
- The website must be a single-page Landing Page.

After that, create a complete website blueprint, including:

Project name / Concept / description
Sections with description, purpose, and word count
(like Hero, Entry, Fragments, Divider, Featured, Explore, Closing, Footer)
Color palette (hex codes) and usage per element
(primary background, text, buttons, accents, highlights)
Fonts / typography (headings, body, sizes, line height)
Layout and alignment details
Responsive design guidelines
Animations / interactivity ideas (hover, fade-in, etc.)
Background / imagery suggestions
Accessibility considerations

Also generate a grid image showing all palette colors, with each square labeled with its hex code on top.

Output everything clearly like a blueprint, ready to implement in code.
Do not use "Digital Diary" — create a new project name and sections based on the niche and palette.
